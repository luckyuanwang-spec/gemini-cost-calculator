# Gemini API Cost Calculator

Free online tool to estimate and compare Google Gemini API costs. Covers all Gemini models including the latest Gemini 3.5 Flash, 2.5 Pro, and more.

🔗 **Live Demo**: [https://gemini-cost-calculator.vercel.app/](https://gemini-cost-calculator.vercel.app/)

## Features

- 💰 **Cost Calculator** — Estimate monthly API spending by input/output tokens and request volume
- 📊 **Cross-Provider Comparison** — Compare costs with OpenAI (GPT-4o, GPT-4.1, o3) and Anthropic (Claude)
- 📋 **Full Pricing Table** — Complete Gemini pricing for text, image, audio, and embedding models
- 🏷️ **Tier Selector** — Standard, Batch (50% off), and Priority (1.8x) pricing
- 🌐 **Bilingual** — English / 中文 toggle
- 📱 **Responsive** — Works on desktop and mobile
- 🔍 **SEO Optimized** — Schema.org markup, meta tags, sitemap
- ❓ **FAQ** — Common questions about Gemini API pricing
